# 🦷 بوبيان — نظام المحاسبة الذكي
## دليل التثبيت والتشغيل على Windows

---

## الخطوة 1 — تحميل Node.js

1. افتح المتصفح واذهب لـ: https://nodejs.org
2. اضغط على **"LTS"** (الزر الأخضر)
3. شغّل الملف المحمّل واتبع خطوات التثبيت (Next → Next → Install)
4. بعد التثبيت، افتح **Command Prompt** واكتب:
   ```
   node --version
   ```
   لازم يطلع رقم مثل: v20.x.x ✅

---

## الخطوة 2 — نسخ ملفات النظام

1. أنشئ مجلد جديد على سطح المكتب اسمه: **boubyan**
2. ضع فيه الملفات التالية:
   - server.js
   - package.json
   - مجلد public (فيه index.html)

---

## الخطوة 3 — تثبيت المكتبات

1. افتح مجلد **boubyan** (الذي أنشأته)
2. في شريط العنوان بالأعلى، اكتب **cmd** واضغط Enter
3. اكتب الأمر التالي واضغط Enter:
   ```
   npm install
   ```
4. انتظر حتى يكتمل (دقيقة واحدة تقريباً)

---

## الخطوة 4 — تشغيل النظام

في نفس نافذة cmd، اكتب:
```
node server.js
```

لازم يظهر:
```
✅ بوبيان للمحاسبة - يعمل على http://localhost:3000
📂 البيانات محفوظة في: ...database.json
```

---

## الخطوة 5 — فتح النظام

افتح المتصفح واكتب:
```
http://localhost:3000
```

🎉 النظام جاهز!

---

## الخطوة 6 — تشغيل تلقائي عند بدء Windows (اختياري)

1. اضغط **Win + R** واكتب: `shell:startup`
2. أنشئ ملف نصي اسمه **boubyan.bat** وضع فيه:
   ```
   @echo off
   cd C:\Users\[اسم المستخدم]\Desktop\boubyan
   node server.js
   ```
3. احفظ الملف في مجلد Startup

من الآن، النظام يشتغل تلقائياً مع Windows!

---

## إعداد تيليجرام

1. افتح تيليجرام وابحث عن: **@BotFather**
2. أرسل: `/newbot`
3. اختر اسم للبوت (مثلاً: Boubyan Accounting)
4. انسخ الـ **Token** الذي يعطيك إياه
5. في النظام، اذهب لتبويب **تيليجرام**
6. الصق الـ Token واضغط **تشغيل البوت**
7. ابعث `/start` للبوت وابدأ!

### الأوامر المتاحة على تيليجرام:
- `/stats` — إحصائيات الشهر الحالي
- `/comm` — عمولات الدكاترة
- `/pending` — العمولات غير المدفوعة
- **ابعث ملف Excel** — يتحفظ تلقائياً
- **أي سؤال** — AI يجاوبك

---

## استخدام النظام يومياً

### رفع تقرير شهري:
1. من البرنامج الطبي: **Reports → Reception → Daily Income**
2. صدّر Excel
3. في النظام: **رفع البيانات** → اسحب الملف
4. ✅ البيانات تتحفظ تلقائياً

### أو من تيليجرام:
1. ابعث الملف مباشرة للبوت
2. يتحفظ تلقائياً بدون ما تفتح الكمبيوتر

---

## مكان حفظ البيانات

البيانات محفوظة في:
```
boubyan/data/database.json
```

**نصيحة:** انسخ هذا الملف على OneDrive أو USB كنسخة احتياطية أسبوعياً.

## في حال وجود مشكلة

**المشكلة:** cmd يقول "node is not recognized"
**الحل:** أعد تثبيت Node.js وأعد فتح cmd

**المشكلة:** المتصفح يقول "لا يمكن الوصول للموقع"
**الحل:** تأكد أن server.js شغّال في cmd

**المشكلة:** الملفات لا ترفع
**الحل:** تأكد أن اسم الملف يحتوي على "DailyIncome" أو "Payments"
